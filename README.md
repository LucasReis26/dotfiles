# Dotfiles

## Descrição

Esse repositório tem um propósito bem claro de guardar as principais configurações que eu utilizo ou quem sabe venha utilizar daqui a um tempo. Além das configurações deixarei documentado como você pode utilizar essas mesmas configs no seu computador!

## Dependências.

- Utilizar uma distro linux.
    - 99% das configurações presentes nesse repositório são de softwares presentes em distros linux, você pode utilizar eles também no windows via [WSL](https://learn.microsoft.com/pt-br/windows/wsl/install), mas definitivamente não é o propósito inicial.
- [Stow](https://www.gnu.org/software/stow/)
    - Utilizado para criar links simbólicos de forma facilitada em distros linux.
